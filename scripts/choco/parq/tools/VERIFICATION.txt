Verification is intended to assist the Chocolatey moderators and community

BACKGROUND
The embedded software have been built from source held at <https://github.com/elastacloud/parq>
The full build script is hosted here: <https://github.com/elastacloud/parq/blob/master/scripts/choco/package.ps1>
The build is executed on a hosted file server in Elastacloud 
The build process created the file parqInstall.zip which is stored as a release on Github https://github.com/elastacloud/parq/releases with a correct tag https://github.com/elastacloud/parq/releases/tag/3.1.1.0-alpha

VERIFICATION PROCESS
1. Download the official released file https://github.com/elastacloud/parq/releases/download/3.1.1.0-alpha/parqInstall.zip
2. Check the file hash is as per this document. This zip file should have a SHA256 checksum of 97E2500F77B2B174780027069E4261C625E456542DADF7BCA01E9ED402216E4D
3. Extract the bundled parqInstall.zip from the chocolatey package, has with get-filehash and compare to the hash 97E2500F77B2B174780027069E4261C625E456542DADF7BCA01E9ED402216E4D and the hash derived from 2 above
4. If all three hashes match (hash from github, hash from chocolatey, hash from build process), the file is verified.
