VERIFICATION PROCESS
1. Download the official released file https://github.com/elastacloud/parq/releases/download/3.1.1.0/parqInstall.zip
2. Check the file hash is as per this document. This zip file should have a SHA256 checksum of A55DF33F8E8C49DCD0BF81C301F9D154F94625F48C2AD8281729EC3C171299AE
3. Extract the bundled parqInstall.zip from the chocolatey package, has with get-filehash and compare to the hash A55DF33F8E8C49DCD0BF81C301F9D154F94625F48C2AD8281729EC3C171299AE and the hash derived from 2 above
4. If all three hashes match (hash from github, hash from chocolatey, hash from build process), the file is verified.
