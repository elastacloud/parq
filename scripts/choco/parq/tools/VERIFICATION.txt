Verification is intended to assist the Chocolatey moderators and community

The embedded software have been built from source held at <https://github.com/elastacloud/parq>

The build process created the file parqInstall.zip which is stored as a release on Github https://github.com/elastacloud/parq/releases with a correct tag https://github.com/elastacloud/parq/releases/tag/2.0.0.0

VERIFICATION PROCESS
1. Download the official released file https://github.com/elastacloud/parq/releases/download/2.0.0.0/parqInstall.zip
2. Check the file hash is as per this document. This zip file should have a SHA256 checksum of 937F170934ECC9315ABA71C2B0A7476409B360E75FB8D2316E6B7CDC5D53EC8C
3. Extract the bundled parqInstall.zip from the chocolatey package, has with get-filehash and compare to the hash 937F170934ECC9315ABA71C2B0A7476409B360E75FB8D2316E6B7CDC5D53EC8C and the hash derived from 2 above
4. If all three hashes match (hash from github, hash from chocolatey, hash from build process), the file is verified. 

EXAMPLE: 
wget https://github.com/elastacloud/parq/releases/download/2.0.0.0/parqInstall.zip -OutFile .\parqInstall.zip 
(get-filehash .\parqInstall.zip).Hash -eq "937F170934ECC9315ABA71C2B0A7476409B360E75FB8D2316E6B7CDC5D53EC8C"
